# teja2.com - GitHub Pages (SEO + AI JSON)

Este paquete está optimizado para GitHub Pages y contiene:
- SEO técnico por página (title, meta description, canonical, OG/Twitter, JSON-LD).
- `sitemap.xml`, `robots.txt`, `CNAME`.
- Imágenes locales en `assets/*.webp`.
- JSON enriquecido para agentes IA:
  - global: `/ai/content.json`
  - por página: `/<ruta>/content.json`

## Publicación
1. Sube esta carpeta (`site/`) a un repo nuevo.
2. En GitHub: Settings -> Pages -> Deploy from branch `main` root.
3. En DNS, apunta `www.teja2.com` a GitHub Pages.
